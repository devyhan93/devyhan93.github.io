#️⃣ 2020-04-08 Til iTerm Hexo oh-my-zsh

학습내용 / 회고

  • 맥에서 사용하던 기본 터미널 대신에 iTerm이라는 프로그램을 설치하여 사용해보았다 oh-my-zsh을 사용하여 더 멋진 쉘을 구성해보았다.
  • 깃헙의 기본 add, commit, push에 대하여 자세히 배울 수 있었다. 과정을 택배에 비유하여 add를 하면 나의 코드가 포장되고, commit을 하면 송장에 정보를 기입하고, push 하면 발송을 하는 설명이 인상깊었다.
  • Hexo라는것을 알게 되어 적용시켜 보았다. 개인 블로그를 Git과 연동하여 이렇게 빠르게 구현할 수 있다는게 정말 놀라웠다.

해야할 일

  • Hexo 테마 변경해보기
  • Git 명령어와 친숙해지기
  • 현재의 깃 저장소에 있는 정리 문서 구조와 Hexo 구조를 병합하여 좋은 구조 생각해보기
