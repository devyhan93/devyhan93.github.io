1.UIKit

UIKit은 iOS, tvOS의 앱을 위한 코코아 터치 레이어에 포함되어있는 사용자의 인터페이스를 관리하고, 이벤트 처리를 하는것이 주 목적인 프레임워크이다.

코코아 터치 레이어는 Mac OS X cocoa API를 기반으로 다양한 프레임워크가 있다

Frameworks use
UIKit UI 요소, 라이프 사이클 관리, 제스처, 터치
Address Book 연락, 편집
Event Kit 캘린더 일정
Game Kit 게임 센터, p2p 네트워킹
iAd 애플의 광고 플랫폼
Map Kit 애플 맵, 구글 맵
Message Kit 이메일, SMS
