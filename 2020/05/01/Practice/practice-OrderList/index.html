Development info

Code Storyboard
✔ ✔
  • 주문 버튼 터치 시 각 아이템에 대한 수량 증가
  • 각 아이템별 주문 수량 증가에 따른 결제금액 증가
  • 결제 진행 시 소지금액에서 결제금액만큼 차감 및 주문수량 초기화
  • 소지금액이 결제금액보다 적은 경우 결제 불가 메시지 Alert 띄우기
  • 초기화 버튼은 주문수량, 결제금액, 소지금액을 모두 초기화
  • 초기화 버튼 이외에 한 번 올린 주문수량을 내리는 기능은 없음
